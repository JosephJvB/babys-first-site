Scope in Javascript

Scope in javascript determines which parts of the code have access to a function or variable. The scope of a javascript item can be either local or global. If a variables value is defined globally, then anytime that variable is referenced, the global value will be applied. Local scope means the value of a variable will defined only for the block of code that it sits in. If a global variable is redefined locally, then the value of the local variable will apply to the local code, while the global value will apply to the rest of the code.
