Javascript

In the JavaScript language, describe all the use cases for each of:
  • parentheses () - e.g. '1. Execute a function var result = doSomething()'

  • Parentheses are used in functions to contain the parameter(s) of the function. eg: function myFunction(parameter1) {}. Parentheses are also used to control the order that a section of code is read by the DOM: ((a && b) || (c && d)).

  • brackets []

    • Brackets are used to contain items in an array-type variable. var myArray = ["item1", "item2"]. When want to access an item in an array you must use brackets to hold the array-number of the item that you want to access: myArray[1] = item2 Brackets are also used when stating the properties of an object in bracket notation. Bracket notation looks like this: myObject["property"]

  • braces {}

    • Curly braces are used when coding a function. In a function the curly braces job is to hold a block of code that is excecuted when the function is called. function myFunction () { Run this code :D } Curly braces are also used when creating objects, they contain the properties and values of the object. var myObj = { property: "value", property: "value" }

  • single quotes ''
  • double quotes ""

    • As far as I know, single and double quotes can be used interchangably. They are both used to contain a string element. ie: a word value that you want JS to recognise that is not native to the program like var or function.
